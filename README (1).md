# 🎧 Spotify UI Clone (Figma Design)

## 📌 Project Overview
This project is a Spotify UI Clone designed in Figma, focused on replicating the modern and clean interface of the Spotify application.

The goal of this project is to practice UI/UX design principles including layout structuring, color systems, typography, and prototyping.

---

## 🔗 Figma Design Link
https://www.figma.com/design/KZPX4gWdoihtNsVa2oP2rc/Spotify-UI-Clone

---

## 🎯 Objectives
- Recreate Spotify’s user interface
- Understand modern UI/UX patterns
- Practice wireframing and prototyping
- Improve design consistency and usability

---

## 🧩 Features & Components
- Home Screen
- Sidebar Navigation (Home, Search, Library)
- Playlist Section
- Music Player Controls
- Track Information Display
- Volume Control
